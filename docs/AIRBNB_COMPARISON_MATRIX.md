# StayHia vs Airbnb Comparison Matrix (Public Pattern Parity)

> Goal: Match Airbnb-level fundamentals, then exceed with speed, clarity, and trust UX.

## 1) Discovery & Search
- Airbnb pattern:
  - Where / When / Who primitive
  - Flexible dates, map + filters
- StayHia parity:
  - Same primitive model
  - Same core filters (price, property type, amenities, cancellation, rating)
- StayHia improvement:
  - Faster filter presets by traveler intent (work, family, long-stay)
  - “Price confidence” indicator (historical trend band)

## 2) Listing Card + Detail
- Airbnb pattern:
  - Card: image, location, rating, total price, badges
  - Detail: gallery, amenities, map, host profile, rules
- StayHia parity:
  - Full card + detail structure
- StayHia improvement:
  - More explicit cleaning/safety standards labels
  - Better anti-surprise fee section before checkout

## 3) Booking Procedure
- Airbnb pattern:
  - Availability check -> price breakdown -> reserve/pay
  - Instant book or request-to-book
- StayHia parity:
  - Same booking state machine + hold windows
- StayHia improvement:
  - Clearer timeline panel (request sent, host response ETA, auto-expire)

## 4) Trust & Safety
- Airbnb pattern:
  - Reviews, identity checks, host/guest policies, dispute handling
- StayHia parity:
  - Reviews + verification + moderation + dispute center
- StayHia improvement:
  - Unified trust score with explainable factors
  - Better fraud warnings before high-risk bookings

## 5) Host Operations
- Airbnb pattern:
  - Listing wizard, calendar, dynamic pricing, payouts
- StayHia parity:
  - Same host command center
- StayHia improvement:
  - Listing quality AI assistant (title/photo/amenities score)
  - Faster bulk calendar rules

## 6) Payments & Payouts
- Airbnb pattern:
  - Service fees + taxes + payout delays
- StayHia parity:
  - Country-aware breakdown, payout schedules
- StayHia improvement:
  - More transparent fee explainer and payout ETA tracker

## 7) Notifications
- Airbnb pattern:
  - Push/email/SMS for booking lifecycle
- StayHia parity:
  - Same lifecycle notifications
- StayHia improvement:
  - Priority levels with quiet hours + locale-aware templates

## 8) Legal/Policy
- Airbnb pattern:
  - Terms, privacy, cancellation policies, refund rules
- StayHia parity:
  - Full legal page set + policy versioning
- StayHia improvement:
  - More readable plain-language summaries per policy
