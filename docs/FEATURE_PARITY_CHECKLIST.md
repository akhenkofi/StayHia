# StayHia Feature Parity Checklist (Airbnb-level)

## Guest Side
- [ ] Global search (Where / When / Guests)
- [ ] Flexible dates + map search
- [ ] Filters (price, type, amenities, ratings, cancellation)
- [ ] Listing detail page (gallery, amenities, host profile, house rules)
- [ ] Availability calendar + instant book/request to book
- [ ] Transparent pricing breakdown (nightly, cleaning, service fee, taxes)
- [ ] Wishlist save/share
- [ ] Booking management (upcoming/past/cancel)
- [ ] In-app messaging with host
- [ ] Reviews (write + read)

## Host Side
- [ ] Host onboarding + identity/business verification hooks
- [ ] Listing creation wizard (title, photos, location, amenities, pricing)
- [ ] Calendar/availability controls
- [ ] Pricing controls (base, weekend, seasonal, min/max stay)
- [ ] Reservation inbox + accept/decline rules
- [ ] Earnings dashboard + payout setup
- [ ] Co-host support
- [ ] Review response tools

## Admin/Operations
- [ ] User/listing moderation queue
- [ ] Fraud/risk rules and flags
- [ ] Disputes and chargeback handling
- [ ] KYC/KYB review tools
- [ ] Payout reconciliation panel
- [ ] Platform analytics (GMV, bookings, conversion, retention)

## Platform Systems
- [ ] Auth (email/phone/social)
- [ ] Role/permission model (guest/host/admin/support)
- [ ] Payments + payouts (multi-country)
- [ ] Notifications (email, push, sms)
- [ ] Localization + currency display
- [ ] Legal (TOS, Privacy, refund, cancellation)
- [ ] Tax and invoice support (region aware)

## StayHia Improvements Over Baseline
- [ ] Faster host onboarding (guided quality checker)
- [ ] AI-powered listing quality score
- [ ] Dynamic fraud + reputation signals
- [ ] Better transparent pricing UX
- [ ] Multi-lingual support from day one
- [ ] Unified support/dispute timeline for users
