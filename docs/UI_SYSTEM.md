# StayHia Interface System (Initial)

## Color System
- Primary: #0F4C81 (trust blue)
- Primary Hover: #0C3E69
- Accent: #FF5A5F (booking CTA)
- Success: #16A34A
- Warning: #F59E0B
- Danger: #DC2626
- Background: #F8FAFC
- Surface: #FFFFFF
- Border: #E2E8F0
- Text Main: #0F172A
- Text Secondary: #475569

## Typography
- Headline: 700, tight tracking
- Body: 400/500
- Scale: 12/14/16/20/24/32

## Core Components
- Search bar (Where/When/Who)
- Listing card
- Price breakdown card
- Booking CTA block
- Host profile panel
- Review card
- Notification toast

## UX Principles
- Fee transparency early
- One primary action per screen
- Keep booking confidence visible (rating, cancellation rule, host response stats)
- Reduce surprise and ambiguity
