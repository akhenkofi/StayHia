# Interface Color and Visual Notes

## Brand Positioning
- Premium trust + global accessibility
- Modern hospitality without over-dark themes

## Accessibility Targets
- WCAG AA contrast minimum
- Keyboard focus visibility on all CTAs
- Color not sole indicator for status

## Suggested Theme Variants
- Default: Trust Blue + Coral CTA
- Night: Deep Slate + Warm Coral
- High contrast mode for accessibility

## Motion
- Subtle transitions (<180ms)
- No excessive animation in booking/payment screens
