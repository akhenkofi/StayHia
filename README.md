# StayHia — Stay Here. Stay Anywhere

Global accommodation marketplace inspired by Airbnb’s proven model, with feature parity targets and strategic improvements.

## Core Principle
Build from public UX patterns and first-principles product design.
Do **not** copy proprietary code, assets, or private data.

## Monorepo Layout
- `web/` — guest + host + admin frontend
- `api/` — backend APIs, auth, booking, payments, payouts, messaging, reviews
- `docs/` — PRD, feature parity checklist, architecture, launch plan
- `scripts/` — utility scripts

## Launchers
Use these one-click scripts from Finder:
- `Start StayHia.command`
- `Stop StayHia.command`
- `Status StayHia.command`

## Phase Plan (High-Level)
1. Foundation: auth, roles, listing CRUD, search, calendar availability, booking flow
2. Monetization: payments, fees, payouts, coupons, cancellation/refund
3. Trust: reviews, verification, moderation, disputes, fraud controls
4. Scale: analytics, notifications, growth loops, localization, legal hardening
